# Проектирование прецедентов использования


![uc_diagram](https://user-images.githubusercontent.com/80625335/140235178-14892490-18b4-43b0-9fae-ef9bb9ed5a8a.png)

## Список акторов:
+ Неавторизованный пользователь (не имеет прав ни на что, кроме регистрации и авторизации)
+ Авторизованный пользователь (имеет доступ к основному функционалу)
+ Админ (авторизованный пользователь, который может банить и сильно влиять на рейтинг обычных пользователей)
+ Суперадмин (админ, который дополнительно может производить рассылки важных сообщений, разжаловать админов, редактировать информацию об общежитии)
